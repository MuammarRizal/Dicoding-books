# Latihan Front-end Development

### Dokumentasi

1. Clone repository ini

```bash
    git clone https://github.com/MuammarRizal/Dicoding-books.git
```

2. Masuk ke folder Dicoding-Books

```bash
    cd Dicoding-Books
```

3. Install Dependencies

```bash
    $ npm install
```

4. Build App

```bash
    $ npm run build
```

5. Jalankan dalam mode Development

```bash
    $ npm run start-dev
```
